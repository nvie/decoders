## httpsUrl

<DecoderSig name="httpsUrl" type="URL" source="strings.ts#L88-L95" />

Accepts URL strings and URL instances, but only HTTPS ones. Returns the value as a URL
instance.

<DecoderPlayground
  decoder="httpsUrl"
  examples={[
    "'https://nvie.com:443'",
    "'http://nvie.com'",
    "'git+ssh://github.com/nvie/decoders.git'",
  ]}
/>

**Tip!** If you need to limit URLs to different protocols than HTTP, you can do as the
HTTPS decoder is implemented: by adding further conditions using a `.refine()` call.

```ts
import { url } from 'decoders';

const gitUrl: Decoder<URL> = url.refine(
  (value) => value.protocol === 'git:',
  'Must be a git:// URL',
);
```

## nanoid

<DecoderSig
  name="nanoid"
  params="{ size: number }"
  type="string"
  source="strings.ts#L106-L115"
/>
<DecoderSig
  name="nanoid"
  params="{ min?: number; max?: number }"
  type="string"
  source="strings.ts#L106-L115"
/>

<Since version="2.4" />

Accepts and returns [Nano ID](https://zelark.github.io/nano-id-cc) string values. By
default, expects a standard 21-char nanoid, but you can optionally specify different size
constraints. It assumes the default nanoid alphabet. If you're using a custom alphabet,
use [`regex()`][] instead.

<DecoderPlayground
  decoder="nanoid()"
  examples={[
    "'1-QskICa3CaPGcKuYYTm1'",
    "'vA4mt7CUWnouU6jTGbMP_'",
    "'123E4567E89B12D3A456426614174000'",
    "'abcdefghijkl'",
    "'$*&(#%*&('",
  ]}
/>

## Limiting string size

<DecoderSig
  name="sized"
  params="decoder: Decoder<string>, limit: { size: number }"
  type="string"
  source="misc.ts#L35-L44"
/>
<DecoderSig
  name="sized"
  params="decoder: Decoder<string>, limit: { min?: number; max?: number }"
  type="string"
  source="misc.ts#L35-L44"
/>

<Since version="2.9" />

Takes any string decoder and rejects values that don't match the given size constraints.
Also works with [arrays][sized-arrays] and [sets][sized-sets].

`SizeOptions` accepts `min`, `max`, or `size` (shorthand for exact length).

<DecoderPlayground
  decoder="sized(string, { min: 2, max: 10 })"
  examples={["'hello'", "'hi'", "'x'", "'this string is too long'", '42']}
/>
