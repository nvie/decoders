## httpsUrl

<DecoderSig name="httpsUrl" type="URL" source="strings.ts#L77-L84" />

Accepts strings that are valid URLs, but only HTTPS ones. Returns the value as a URL
instance.

<DecoderPlayground
  decoder="httpsUrl"
  examples={[
    "'https://nvie.com:443'",
    "'http://nvie.com'",
    "'git+ssh://user@github.com/foo/bar.git'",
  ]}
/>

**Tip!** If you need to limit URLs to different protocols than HTTP, you can do as the
HTTPS decoder is implemented: by adding further conditions using a `.refine()` call.

```ts
import { url } from 'decoders';

const gitUrl: Decoder<URL> = url.refine(
  (value) => value.protocol === 'git:',
  'Must be a git:// URL',
);
```

## nanoid

<DecoderSig

  source="strings.ts#L95-L104"
