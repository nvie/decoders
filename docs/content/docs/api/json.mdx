## json

<DecoderSig name="json" type="JSONValue" source="json.ts#L26-L49" />

Accepts any value that's a valid JSON value.

In other words: any value returned by `JSON.parse()` should decode without
failure.

```typescript
type JSONValue =
    | null
    | string
    | number
    | boolean
    | { [string]: JSONValue }
    | JSONValue[]
```
