## object

<DecoderSig
  name="object"
  params="{ field1: Decoder<A>, field2: Decoder<B>, ... }"
  type="{ field1: A, field2: B, ... }"
  source="objects.ts#L55-L136"
  multiline
/>

Accepts objects with fields matching the given decoders. Extra fields that exist on the
input object are ignored and will not be returned.

<DecoderPlayground
  decoder="object({ x: number, y: number })"
  examples={['{ x: 1, y: 2 }', '{ x: 1, y: 2, z: 3 }', '{ x: 1 }']}
/>

For more information, see also
[Difference between `object`, `exact`, and `inexact`](#difference-between-object-exact-and-inexact).

## inexact

A "loose object" decoder. Like [`object()`][], but will pass through any extra fields on
the input object unvalidated that will thus be of `unknown` type statically.

## record

<DecoderSig
  name="record"
  params="values: Decoder<V>"
  type="Record<string, V>"
  source="collections.ts#L8-L57"
/>

<DecoderSig
  name="record"
  params="keys: Decoder<K>, values: Decoder<V>"
  type="Record<K, V>"
/>

Accepts objects where all values match the given decoder, and returns the result as a
`Record<string, V>`.

This is useful to validate inputs like `{ [key: string]: V }`.

### Decoding values only

The default call takes a single argument and will validate all _values_. For example, to
validate that all values in the object are numbers:

<DecoderPlayground
  decoder="record(number)"
  examples={['{ red: 1, blue: 2, green: 3 }', "{ hi: 'not a number' }"]}
/>

### Decoding keys and values

If you also want to validate that keys are of a specific form, use the two-argument form:
`record(key, value)`. Note that the given key decoder must return strings.

For example, to enforce that all keys are emails:

<DecoderPlayground
  decoder="record(email, number)"
  examples={["{ 'me@nvie.com': 1 }", "{ 'no-email': 1 }"]}
/>

## pojo

## Difference between `object`, `exact`, and `inexact`

The three decoders in the "object" family are very similar and only differ in how they
treat extra properties on input values.

<DecoderPlayground
  globals={{
    fields: '{ x: number, y: number }',
  }}
  preface={`
    const fields = {
      x: number,
      y: number,
    };
  `}
  decoder={[
    ['object', 'object(fields)'],
    ['exact', 'exact(fields)'],
    ['inexact', 'inexact(fields)'],
  ]}
  examples={['{ x: 1, y: 2, z: 3 }', '{ x: 1, y: 2 }']}
/>
